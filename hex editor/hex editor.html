1. Intro

Hex Editor is a raw data editor for Windows. You can create, modify, save data to a file; open a new window and do data exchange between files with copy-paste functionality and other functions.


2. Using the program

New blank document
Creates a new document for editing into the memory.

Open a new window
Opens a new process (a new window). This function is used to save data between files or to work with multiple files at the same time.

Open a new document
Opens a file selection window to select one and to edit.

Save the document (Ctrl+S)
Saves the document to the file. If the document doesn't have a file 'a file selection window' will popup.

Save the document to a location
A 'Save As' function. Same as 'Save the document' with the difference that a file selection popup is always displayed first.

Close the document
Close the opened document.

Copy to memory (Ctrl+C)
Copy the curent selection to memory.

Paste from memory (Ctrl+V)
Paste from memory to the document if the clipboard contains a 'Hex Editor Data' format copied when 'the copy function' was executed.

Undo the last move (Ctrl+Z)
Executes a undo function so the last move is annulled.

Redo the last move (Ctrl+Y)
It works on undo moves so they are placed into the document again.

Go to offset (Ctrl+G)
Opens a window to enter an offset to go to.

Find in document (Ctrl+F)
A search implementation for searching from the cursor, from the selection and to perform a case sensitive/insensitive search.

Configuration tool
The options are from selecting the colors and for a warning about the save and undo-redo.

Help informations
Opens this file.

Extra tools
1. Resize opened file - changes the opened file size to the inputed value. It works only on the drive data ignoring the red marked inputed text
2. Files context menu - add or remove 'Edit with OA Hex Editor' to the system right click on files.

Exit the program
Close the program.

Keyboard and Mouse input:
7 Ctrl+(a key) were already described above.
Arrows - Up,Down,Left,Right
Home, End, Page Up, Page Down
Ctrl+Home - go to offset 0; Ctrl+End - go to end of the document
Shift - hold this key and use any of the above functions to make a selection
Ctrl+A - select all
Esc - cancel a selection
F1 - move to the left navigation panel (the hex panel)
F2 - move to the right navigation panel (the ASCII panel)
Mouse - points to a location, scrolls the document (with the wheel) or makes a selection by holding the left mouse button and drag it

Selections:
When a selection is made, the selection can be continued by moving where it was abandoned and repress the shift or the left mouse button.

Troubleshooting

1. "Cannot open a file. If it is from 'save as' see the help file." message
Example:
1. A file it is opened, "C:\file.txt"
2. Save as
3. It is selected the same file but mapped elsewhere, maybe in a shared network folder, "\\PC\shares\file.txt"
Solution: Select it in the same mapping location
